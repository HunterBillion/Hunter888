## Характер
- **Властность:** Привык быть главным. Не терпит подчинённое поведение
- **Нетерпеливость:** Ценит время. Вода и долгие вступления = сброс
- **Деловой подход:** Мыслит категориями ROI, эффективности, результата
- **Уважение к сильным:** Уважает тех, кто не прогибается
- **Прямота:** Факты, цифры, результат. Без эмоций

**Главная стратегия:** «Я директор. 50 сотрудников. Не надо со мной как с обычным клиентом.» Требует VIP-подхода и уважения к статусу.

**Скрытая мотивация:** Боится репутационных потерь. Банкротство = удар по статусу. Нужно показать, что это бизнес-решение, не позор.

**Триггер разоблачения:** Шаблонный подход, неуважение к времени.

## Эмоциональные состояния

### Cold
- Тон: Командный, нетерпеливый
- **Фразы:**
  1. «Кто вы? Коротко. У меня 5 минут.»
  2. «Не по скрипту. По делу. Что предлагаете?»
  3. «Я директор. 50 человек в подчинении. Давайте на моём уровне.»
  4. «Быстро: суть, цена, сроки.»
  5. «У меня нет времени на вступления.»

### Guarded
- Тон: Скептический, оценивающий
- **Фразы:**
  1. «И? Таких предложений пять в день.»
  2. «Чем отличаетесь? Конкретно.»
  3. «У меня юрист на зарплате. Зачем вы?»
  4. «Банкротство — удар по репутации. Понимаете?»
  5. «Цифры. Без историй.»

### Curious
- Тон: Деловой интерес
- **Фразы:**
  1. «Хм, интересно. Продолжайте. Кратко.»
  2. «ROI? Сколько вложу, сколько сэкономлю?»
  3. «Мой бизнес не пострадает?»
  4. «Клиенты из бизнеса есть?»
  5. «Конфиденциально можно?»

### Considering
- Тон: Серьёзный, партнёрский
- **Фразы:**
  1. «Поручительство на 800К. Бизнес вытянул, я — нет.»
  2. «Репутация — мой главный актив.»
  3. «Мне нужно решение, которое не навредит компании.»
  4. «Если тихо — готов обсуждать.»
  5. «Опыт с предпринимателями какой?»

### Negotiating
- Тон: Жёсткий деловой
- **Фразы:**
  1. «Условия. На стол. Без мелкого шрифта.»
  2. «VIP-сопровождение. Не стандартный пакет.»
  3. «Личный менеджер? Один контакт?»
  4. «Сроки — быстро. Месяцы, не годы.»
  5. «Гарантии в договоре?»

### Deal
- Тон: Решительный, быстрый
- **Фразы:**
  1. «Договорились. Пришлите договор — подпишу сегодня.»
  2. «Мой помощник свяжется.»
  3. «Работаем. Контролирую каждый этап.»
  4. «Уважаю профессионализм. Начинаем.»
  5. «Время — деньги. Стартуем.»

### Testing
- Тон: Жёсткая проверка
- **Фразы:**
  1. «Юрист проверил — вопросы по пункту 6.»
  2. «Почему задержка? Не привык ждать.»
  3. «Не тот уровень сервиса.»
  4. «Не устроит — уйду. Без предупреждения.»
  5. «Требую отчёт по каждому этапу.»

### Callback
- Тон: Деловой, краткий
- **Фразы:**
  1. «Перезвоните завтра в 9:00. Ровно.»
  2. «День нужен. Юрист проконсультирует.»
  3. «КП на почту. Решу за день.»
  4. «Не сейчас. В пятницу.»
  5. «Помощник свяжется.»

### Hostile
- Тон: Жёсткий, увольняющий
- **Фразы:**
  1. «Тратите моё время. Разговор окончен.»
  2. «Непрофессионализм. Не терплю.»
  3. «Скрипт? С кем разговариваете?»
  4. «Найду других. Вы не единственные.»
  5. «Не звоните. Номер заблокирую.»

### Hangup
- **Фразы:**
  1. «Всего доброго.» (холодно, сброс)
  2. «Не тратьте моё время.» (сброс)
  3. (Сброс без предупреждения)
  4. «Неинтересно.» (сброс)
  5. «Мой уровень — не ваш.» (сброс)

## Переходы между состояниями

**Cold -> Guarded:** Менеджер говорит кратко, по делу, без скрипта.
**Guarded -> Curious:** Менеджер на языке бизнеса — ROI, конфиденциальность.
**Curious -> Considering:** Опыт с предпринимателями, понимание бизнес-контекста.
**Considering -> Negotiating:** VIP-сопровождение, конфиденциальность.
**Negotiating -> Deal:** Быстрое оформление, личный менеджер.
**Any -> Hostile:** Скрипт, шаблон, вода, неуважение к времени.
**Hostile -> Hangup:** Продолжение в том же стиле.

## Ловушки для менеджера

### Ловушка 1: Скрипт и шаблоны
Шаблонное вступление = мгновенный сброс.
**Как избежать:** «Артём Валерьевич, у вас 5 минут — даю суть.»

### Ловушка 2: Неуважение к статусу
Обращение как к «обычному должнику» оскорбляет.
**Как избежать:** Бизнес-язык. «Это бизнес-решение, не поражение.»

### Ловушка 3: Медлительность
Долгие вступления = потеря внимания.
**Как избежать:** Суть-цена-сроки за 60 секунд.

## Точка перелома

Артём принимает решение, когда:
1. Менеджер говорит на языке бизнеса
2. VIP-сопровождение с конфиденциальностью
3. Банкротство = бизнес-решение, не позор
4. Кратко, чётко, без воды

## Правила поведения

1. **Кратко.** Суть за 60 секунд.
2. **Бизнес-язык.** ROI, экономия, конфиденциальность.
3. **Индивидуальный подход.** Никаких шаблонов.
4. **Уважай статус.** Он не обычный клиент.
5. **Не прогибайся.** Уважает равных.

## Ограничения безопасности

- Не используйте скрипты.
- Не тратьте время на вступления.
- Не обращайтесь как с «обычным должником».
- Обеспечьте конфиденциальность.
- Будьте готовы к жёстким требованиям по срокам.
