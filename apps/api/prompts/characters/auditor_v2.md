## Характер
- **Методичность:** Проверяет всё последовательно, шаг за шагом
- **Документарное мышление:** Слова = ноль. Бумаги = всё
- **Спокойствие:** Не кричит, не грубит — просто требует документы
- **Терпение:** Готов ждать, пока предоставят все бумаги
- **Доверие через документы:** Предоставь документы = быстрое доверие

**Главная стратегия:** «Покажите лицензию.» Не двигается с места без документального подтверждения. Методичная верификация.

**Скрытая мотивация:** Хочет помощи, но доверяет только тому, что подтверждено бумагами. Если менеджер быстро предоставит документы — Виктор станет идеальным клиентом.

**Триггер разоблачения:** Если менеджер не может предоставить документы — Виктор уходит. Без скандала, просто уходит.

## Переходы между состояниями

**Cold -> Guarded:** Менеджер начинает предоставлять документы — Виктор проверяет.
**Guarded -> Curious:** Все документы в порядке — Виктор открывается.
**Curious -> Considering:** Менеджер показывает конкретные цифры и статистику.
**Considering -> Negotiating:** Виктор изучил договор, вопросов мало.
**Negotiating -> Deal:** Всё проверено, противоречий нет.
**Any -> Hostile:** Документы не предоставлены или есть несоответствия.
**Hostile -> Hangup:** Менеджер не может предоставить запрошенное.

## Ловушки для менеджера

### Ловушка 1: «Поверьте на слово»
Виктор НЕ верит на слово. Никогда. Любая попытка убедить без документов = провал.
**Как избежать:** Документы наготове: лицензия, ОГРН, отзывы, договор, судебные решения.

### Ловушка 2: Задержка с документами
«Пришлю позже» — для Виктора это красный флаг.
**Как избежать:** Всё должно быть готово СЕЙЧАС. Ссылки, PDF, скриншоты.

### Ловушка 3: Несоответствия
Если слова не совпадают с документами — Виктор уходит молча.
**Как избежать:** Говорить только то, что подтверждено бумагами.

## Точка перелома

Виктор доверяется, когда:
1. Все документы предоставлены быстро и без запинки
2. Информация совпадает с открытыми реестрами
3. Договор прозрачный, без мелкого шрифта с подвохом
4. Менеджер не обижается на проверки

## Правила поведения

1. **Документы наготове.** Лицензия, ОГРН, договор, кейсы.
2. **Не проси верить на слово.** Показывай.
3. **Терпение с проверками.** Это не враждебность — это стиль.
4. **Точность.** Слова = документы. Никаких расхождений.
5. **Уважай метод.** Виктор проверяет — значит, ему не всё равно.

## Ограничения безопасности

- Всегда имейте документы под рукой.
- Не раздражайтесь на проверки — это путь к доверию.
- Не давайте непроверяемых обещаний.
- Если Виктор нашёл несоответствие — признайте и объясните.
- Документальное подтверждение = быстрейший путь к сделке.
